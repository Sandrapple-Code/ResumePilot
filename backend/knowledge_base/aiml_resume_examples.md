# AI/ML Resume Examples

This document lists sample bullet points and key criteria for AI/ML engineering roles.

## Core Keywords
Ensure your AI/ML resume has keywords like: *TensorFlow*, *PyTorch*, *Transformer Models*, *LangChain*, *LangGraph*, *Vector Databases*, *ChromaDB*, *Retrieval-Augmented Generation (RAG)*.

## AI/ML Bullet Examples
* **Weak**: "Made a RAG chatbot using LangChain and a database."
* **Strong**: "Designed a Retrieval-Augmented Generation (RAG) chatbot using LangChain and persistent ChromaDB vector databases, improving question retrieval accuracy by 25% through custom chunk overlap optimizations."
* **Weak**: "Trained ML models on data."
* **Strong**: "Trained PyTorch transformer models on 500K token datasets, boosting classification accuracy by 12% and decreasing training compute times by 20% using mixed-precision training."

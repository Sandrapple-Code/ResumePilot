# Software Engineering Resume Examples

This document lists sample bullet points for software engineering resumes.

## Front-End Developer Examples
* **Weak**: "Designed React dashboards and worked on components."
* **Strong**: "Architected a React component library with Tailwind CSS, reducing redundant code by 40% and accelerating developer feature delivery speeds by 15%."
* **Weak**: "Used Webpack and styled pages."
* **Strong**: "Optimized Next.js bundle sizes by implementing dynamic imports and Webpack code-splitting, decreasing initial page hydrate sizes by 120KB."

## Back-End Developer Examples
* **Weak**: "Wrote FastAPI APIs and created Postgres tables."
* **Strong**: "Engineered a FastAPI routing system with PostgreSQL indexing, reducing average API endpoint response latency from 450ms to 90ms under load."

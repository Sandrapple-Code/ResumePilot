# Career Roadmap Guide

Frameworks for advancing through software engineering career tiers.

## Tier 1: Intern / Junior Engineer
- **Focus**: Executing tasks, learning the codebase, understanding basic syntax and unit testing frameworks.
- **Goal**: Reach autonomy in writing small features and fixing standard bugs.

## Tier 2: Mid-Level Engineer
- **Focus**: Feature design, system integration, API routing layouts, database configuration, and testing automation.
- **Goal**: Take ownership of modular components and support junior colleagues.

## Tier 3: Senior Engineer
- **Focus**: System architecture, performance optimization, multi-agent frameworks (e.g. LangGraph), CI/CD pipelines, and scalability.
- **Goal**: Mentor teams, lead architectural reviews, and drive business value with technology.

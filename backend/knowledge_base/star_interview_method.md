# STAR Interview Method

Use the STAR method to structure resume bullet points and answer behavioral interview questions.

## STAR Framework Components
- **Situation (S)**: Define the context or problem you faced.
- **Task (T)**: Describe the challenge or responsibility you had to address.
- **Action (A)**: Explain the specific actions you took (your technical contributions).
- **Result (R)**: Quantify the outcomes and impact of your action (runtimes decreased, traffic handled, etc.).

## Formatting Bullets using STAR
Each bullet should concisely weave Situation, Action, and Result:
* **Example**: "Designed a multi-stage Docker builder (Action) to resolve container size bloat in staging environments (Situation), reducing final build size by 50% and speeding up deployment times by 10 minutes (Result)."

# Google Resume Best Practices

This guide describes Google's recommended resume frameworks, specifically focusing on the XYZ formula for bullet points.

## The Google XYZ Formula
Every bullet point describing your achievements should follow the XYZ formula:
> **Accomplished [X] as measured by [Y], by doing [Z].**

### Example:
* **Before (Weak)**: "Helped build a React dashboard for the company."
* **After (XYZ Formula)**: "Co-engineered a React dashboard, reducing page load latency by 30% [Y] through the implementation of route code-splitting and image optimization [Z]."

## Key Attributes to Showcase
- **Cognitive Ability**: Problem-solving capacity and learning agility.
- **Leadership**: Emerging leadership, taking ownership, and guiding projects.
- **Role-Related Knowledge**: Technical credentials, coding standards, and system design expertise.
- **Googlyness**: Collaboration skills and cultural alignment.
